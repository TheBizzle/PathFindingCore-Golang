PathFindingCore-Golang
===============

## What is it?

The building blocks for constructing and testing your own pathfinding algorithm in Golang and visually representing it in ASCII.  Note that I've only ever utilized this library for implementing A*.

For example usage, see [the corresponding Golang A* library](https://github.com/TheBizzle/AStar-Golang).

## Why is it?

I just do this when learning new programming languages.

See [here](https://github.com/TheBizzle/PathFindingCore) and [here](https://github.com/TheBizzle/PathFindingCore-Haskell) and [here](https://github.com/TheBizzle/PathFindingCore-FSharp) for similar examples.
